# Aria Sokhangoo — Portfolio

Live, interactive AI engineering portfolio built as a single HTML file.

## Experience it

> **[→ Live Site](https://epochalypse00.github.io/Aria-AI/aria-neural-portfolio.html)**

## What it is

A 3D neural sphere you travel through as a portfolio. Click the sphere to enter,
fly between planet-nodes at lightspeed, and explore my work at each destination.

- **Sun** — About me
- **Earth** — Resume & download
- **Mars** — Projects
- **Jupiter** — Skills
- **Saturn** — Contact

## Tech

Built entirely in vanilla HTML, CSS, and JavaScript — no frameworks, no build tools.
Three.js for 3D rendering · Web Audio API for ambient music · Fully self-contained single file.
